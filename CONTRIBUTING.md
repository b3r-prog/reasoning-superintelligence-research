# Contributing to Reasoning & Superintelligence Research Hub

Thank you for wanting to contribute! This is a community resource and every contribution matters.

## Quick Start

1. **Fork** this repository
2. **Clone** your fork locally
3. Make your changes (see guidelines below)
4. **Submit a Pull Request**

## What to Contribute

### ✅ We Accept
- Recent papers (2020–present) on reasoning, alignment, interpretability, scaling, multi-agent systems
- High-quality implementation repos (>100 GitHub stars, or exceptional new work)
- Well-defined open problems with context
- New promising research directions with evidence of traction
- Blog posts / newsletters from established researchers or labs
- Corrections to broken links or factual errors
- Better descriptions or categorizations

### ❌ Please Avoid
- Papers without peer review or arXiv hosting (unless from major labs)
- Self-promotional content without demonstrated community interest
- Vague open problems ("make AI better")
- Repos without documentation
- Duplicate entries

## Format Guidelines

### Papers Table Entry
```markdown
| **Paper Title** | YEAR | Author et al. (Institution) | One sentence on key contribution | [arXiv](link) |
```

### Repo Table Entry
```markdown
| [org/reponame](https://github.com/org/reponame) | What it does in one sentence | ⭐ Xk+ |
```

### Open Problem Entry
Follow the existing numbered list format. Include:
- A specific, answerable research question
- Why it matters (1–2 sentences)
- Any existing partial solutions or relevant prior work

## PR Review Process

PRs are reviewed weekly. We prioritize:
1. Factual accuracy
2. Relevance to the repo's scope
3. Formatting consistency
4. Link validity

## Code of Conduct

Be respectful. This is a scientific community resource. Debate ideas, not people.

## Questions?

Open an issue with the label `question`.
